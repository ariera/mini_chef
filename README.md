MiniChef
========

Generate files based on ERB templates and configuration variables, like the Chef template does.


## Installation

Add this line to your application's Gemfile:

    gem 'mini_chef'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mini_chef
